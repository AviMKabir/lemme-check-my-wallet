## A simple way to budget on the fly
------

* 1-factor search
* Simple & fast results
* Premium mode
* Cloud based data storage

## Technologies used
------
* HTML, CSS, Javascript
* jQuery
* Bootstrap
* BotUI
* AJAX
* MySQL
* Node, Sequelize, Express
* Font Awesome
* JAWSDB
* Handlebars
* DialogFlow / API.AI (Automatic Voice to Text)
